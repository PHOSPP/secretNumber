# SecretNumber
This project is an interactive choice-based game developed using HTML, CSS, and JavaScript. The main idea is to allow the player to type a number and try guessing the random number chosen by the program, giving hints when the player miss the answer so he can try again until he gets it right.


## OBJECTIVES
Learn how to use HTML, CSS, JS, Github and Git

## FEATURES
- Random integer number generation
- Guess tip
- Infinite game loop
